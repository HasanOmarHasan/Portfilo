# 10 Top vscode shortcut should every developers knows
اكتر  10 shortcut لازم كل الdevelopers يعرفها في vscode ، البونص اهم واحدة

الshortcuts ساعدتني جدا اوفر وقت في بعض الحجات الي عشان اعملها بالطريقة العادية كنت بلف الرجاء الصالح عشان اعملها 
 

- *️⃣لو عاوز تظهر اخر فايل فتحته او تروح لفايل جوا المشروع (ctrl + P) وتكتب اسم الفايل 

- *️⃣ تظهر ال side bar او الشريط الجانبي او تخفية (ctrl + B ) 
اظهار شريط terminal او اخفاءة (ctrl + J ) او (crtl + ` ) 
ولو عاوز تفتح صفحة terminal جديدة (ctrl + shift + ` )

- *️⃣ تقفل التاب او صفحة الي شغال فيها (ctrl + w)
او لو عاوز تقفل كل التابات
 (Ctrl+K W or Ctrl+K Ctrl+W) 

- *️⃣التنقل مبين التابات المفتوحة (crtl + tab )

- *️⃣ تروح لسطر معين في الملف (ctrl + g ) وتكتب رقم السطر , مفيدة جدا ف الايرور

- *️⃣ في shortcut مشهورة جدا لو بتيجي تتدور ع كلمة او نص ف الكلام (ctrl + f ) ولو عاوز انك تستبدل الكلمة الي حددتها دي او اي كلمة تانية من خلال (ctrl + H )

- *️⃣ التنقل مبين السطر بسرعة بان المؤشر بيقف مرتين عند كل كلمة مرة ف اول الكلمة و مرة ف اخرها مش بيعدي ع كل حرف (ctrl + سهم لمين او شمال )

- *️⃣مسح سطر كامل (shift + ctrl + k )

- *️⃣ لو عاوز تغير في كذا سطر في نفس الوقت في طريقتين
1️⃣ لو المتحدد نفس الكلمة او الحاجة فممكن من (ctrl + d) وهو يبدا يحدد المتطابق الي بعدو او (ctrl + shift + L ) هايحدد كل المتطابقين او المتشابهين مرة واحدة
2️⃣ لو مختلفين فهتحددهم بنفسك و (alt) وتحدد الي بعدو

- *️⃣ تحرك سطر لفوق او تحت (alt + سهم فوق او تحت )
تكرار السطر (alt + shift + سهم فوق او تحت )

- بونص 
انك تفتح مشروع تاني اشتغلت عليه قبل كدة و عاوز ترجعله في الطريقة العادية هتروح للفولدر الي فية المشروع وتسحبه ل vscode او عن طريق cmd ب code . و هايفتح المشروع لكن (ctrl + r ) هايظهر كل المشاريع الي فتحتها او اشتغلت عليها قبل كدة و لما تدوس علي واحد منهم هايبدل مبين المشروع او اي كان الي فاتحه و الي اخترته 

ممكن يكون بعض الاختلافات لما تستخدم ع نظام ماك فممكن تسرش عليها ان كل دا خاص بالويندوز 
و لو تعرف shortcut تانية اكتبها ف التعليقات و قول لنا كام واحدة كانت جديدة عليك
